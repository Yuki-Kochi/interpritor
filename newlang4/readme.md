# BASIC language SyntaxAnalyzer on Java


## 概要

- STEP2 構文解析　までで必要となるクラス郡。
- 動作には必ずSTEP1で作成後にアップデートされたnewlang3パッケージが必要となる。
- newlang5作製に伴い、newlang4までの実行であってもnewlang5パッケージが必要となった。

## 備考

- 実行には一つ上の階層のnewlang4Mainクラスを利用する。
- 途中で何らかのエラーが発生した際には以下のように処理している。
　ここにはまだ改善の余地があり、本来は途中でエラーがあっても構文解析を続行できるべきである。
	- 入力ファイル中の記述が不適切な場合には、SyntaxExceptionをスローする
	- 本来ありえない場面において(事前にisMatchで確認していないなど)関数が呼ばれたなどの場合には、InternalErrorをスローする。
	- どちらの場合にも発生個所が分かるようエラーメッセージを付記しているが、メッセージの書き方に統一感がないなど問題がある。
	- どちらも、Mainクラスでまとめてキャッチする。
